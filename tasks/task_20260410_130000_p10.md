# Phase 10: 定期タスク実装

## ゴール

タスクの完了フラグ＋全チェックボックスが指定周期で自動リセットされる「定期タスク」機能を実装する。

## 仕様

**概念**
定期タスク = 完了フラグ＋全チェックボックスが指定周期でリセットされるタスク

**メタデータ（`.kobito/tasks/*.json`）**
```json
{
  "repeat_enabled": true,
  "reset_interval": "daily",
  "reset_time": "09:00"
}
```

| フィールド | 説明 |
|-----------|------|
| `repeat_enabled` | 定期リセットの有効/無効（省略時 `true`） |
| `reset_interval` | `every_check` / `hourly` / `daily` / `weekly` / `monthly` |
| `reset_time` | `hourly`以上で使用（`HH:MM` 形式、hourlyは `:MM`） |
| `reset_weekday` | `weekly`のみ（`monday`〜`sunday`） |
| `reset_monthday` | `monthly`のみ（1〜31） |

`reset_interval` フィールドが存在する = 定期タスク。
